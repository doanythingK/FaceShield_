# FaceShield 사용설명서

## 기본 워크플로우
1) 홈 화면에서 영상 파일을 선택합니다.
2) 자동 모자이크 또는 수동 모자이크를 선택합니다.
3) 자동 모드는 "자동 완료 후 파일 저장" 체크 여부에 따라 동작이 다릅니다.
   - 체크함: 자동 저장만 하고 워크스페이스로 이동하지 않습니다.
   - 체크 안 함: 처리 완료 후 워크스페이스로 이동합니다.
4) 워크스페이스에서 프레임을 확인하고 필요하면 브러시/지우개로 보정합니다.
5) 저장 버튼으로 최종 영상을 내보냅니다.

## 자동 모드
- 자동 모자이크 실행 중 취소해도 진행 상태가 저장됩니다.
- 다음에 자동 모드로 들어오면 이어서 실행할지 묻습니다.
- 일부 자동 옵션은 속도가 빨라지는 대신 검출 정확도가 낮아질 수 있습니다.

## 수동 모드
- 브러시/지우개로 직접 마스크를 편집합니다.
- 브러시 크기는 도구 패널에서 조절합니다.

## 출력
- 저장 시 원본 해상도(예: 4K)를 유지합니다.
- 파일명은 원본 이름에 `_blur`가 붙습니다.
- 저장되는 디렉토리는 원본과 동일합니다.

## 단축키 (워크스페이스)
- 다음 이상 프레임: `E`
- 이전 이상 프레임: `Q`
- 프레임 이동: `←` `→` (Shift와 함께 누르면 10프레임 이동)
- 1초 이동: `↑` `↓`
- 처음/끝: `Home` `End`
- 재생/정지: `Space`

## macOS에서 앱 실행(내부 테스트용)
macOS는 서명되지 않은 앱을 차단할 수 있습니다. 아래 절차로 임시 서명 후 실행하세요.

1) 다운로드한 `FaceShield.app`가 있는 폴더로 이동
2) `macos-sign-local.command`를 더블클릭해 임시 서명을 실행
3) `FaceShield.app`를 실행하면 됩니다.

실행 중 오류 로그를 확인하려면:
- `macos-run-log.command`를 더블클릭해 실행하면 `macos-run.log`에 로그가 저장됩니다.
